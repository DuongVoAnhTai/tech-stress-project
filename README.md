# Tech stress project

# 1. Tải dự án về

- Clone hoặc tải file zip của dự án về máy

# 2. Tải model về

- đã có file data tại thư mục data
- Khi train hãy upload file đó lên

# 3. Cài đặt các thư viện cần thiết

- pip install -r requirements.txt

# 4. Chạy ứng dụng

- streamlit run app.py
- ứng dụng sẽ cung cấp đường dẫn để truy cập local host

# 5. Train model

- Train model bằng Google Colab
- Upload file advanced_stress_mental_health.ipynb lên colab
- Tiến hành training
- Sau khi train thành công thì nó sẽ xuất các sơ đồ và 1 file models_mental_health_analysis.zip
- Tải folder vừa giải nén về và chạy ứng dựng
