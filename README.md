# 2. Tải model về

- đã có file data tại thư mục data

# 3. Cài đặt các thư viện cần thiết

- pip install -r requirements.txt
- pip install plotly

# 4. Chạy ứng dụng

- streamlit run app.py
- ứng dụng sẽ cung cấp đường dẫn để truy cập local host

- Train model bằng Google Colab

- Tiến hành training
- Sau khi train thành công thì nó sẽ xuất các sơ đồ và 1 file models.zip
- Tải folder vừa giải nén về và thay thế bằng models demo
